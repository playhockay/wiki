Hockay is a spectator sport. The fans are the point. Without fans, there are no bets.
Without bets, there are no [Looneys](/guides/economy). Without Looneys,
[Le Council](/lore/lpchuip) cannot fund its ongoing investigation into Unexplained Ice
Phenomena.

**You are essential. You are just not on the ice.**

## Bilingual dimension

Some announcements from [The Almighty Ice](/lore/almighty-ice) or [The Puck](/lore/the-puck)
arrive in French. Some arrive in English. Some arrive in both. A few in neither. The game's
canonical language is **both English and French and neither**.

[JM](/lore/commentators) slides between languages mid-sentence. Nobody remarks on it.

## Contributing to this wiki

This wiki is community-run. You can help:

- **Fix something**—click "Edit this page" at the bottom of any article. GitHub opens the `.mdx` source in its web editor; your change becomes a pull request.
- **Add something**—open a pull request against [the repo](https://github.com/playhockay/wiki). For small additions, GitHub's web editor is enough; for larger ones, clone and run locally.
- **Share what you found**—match a sentence The Puck said years ago to an event. Link a player's Ancient boosts to arena patterns. Map the [Sixth](/lore/the-sixth) output distribution.

See [CONTRIBUTING.md](https://github.com/playhockay/wiki/blob/main/CONTRIBUTING.md) for the full contribution guide.
