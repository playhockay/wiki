## Contribute

Anyone can edit. Two ways:

- **[In your browser](/editor)**—sign in with GitHub, edit pages in a visual editor. Your changes open a pull request automatically.
- **[Via a pull request](https://github.com/playhockay/wiki)**—click "Edit this page" on any article or fork the repo.

See [CONTRIBUTING.md](https://github.com/playhockay/wiki/blob/main/CONTRIBUTING.md) for the house rules.
